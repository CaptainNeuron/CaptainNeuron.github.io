Betrayal - Dark House 2

We develop a roguelike mobile game with procedural generated game maps every time you start the game.

Every time the game is on, players will face different challenges. Your goal will change based on the number of players. Sometimes you need to fight with your friends but sometimes players need to cooperate to fight the monster and explore.

Instruction

Procedural Generation

I made a program to randomly generate the game scene I need. I manage to generate the game scenes and various items, monsters, treasure chests, and other objects.

First, I generate the room space. There are two methods. One is to generate regular rooms and the other one is to generate irregular rooms.

Monster AI

The program that control the monsters are relatively simple. We use automata to control the monsters. Monsters will automatically chase the players when they approach. We use A* search algorithm to help them find their way.
